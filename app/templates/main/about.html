About This Platform

The Analyst Performance Tracker is a custom‑built web application for the Prague Club of Investors to monitor and evaluate the performance of investment analyses conducted by its members.

Purpose

The club maintains a Notion table of who analyzed which stock and when. This platform automatically enriches that data with stock‑price information from Yahoo Finance, calculates the return of each approved analysis, and provides each analyst with a personal dashboard showing their performance metrics.

Key Features

  • Domain‑restricted authentication – only @klubinvestoru.com email addresses can register.
  • Password‑less onboarding – new users receive a secure link to set their password.
  • CSV upload – admins can upload the Notion‑exported schedule; the system parses and stores analyses.
  • AI‑assisted ticker matching – optional DeepSeek API integration helps identify stock tickers.
  • Automatic price fetching – daily stock prices are retrieved from Yahoo Finance.
  • Performance calculation – returns are computed for each approved analysis (“On Watchlist”).
  • Analyst dashboards – individual members see their own performance statistics.
  • Admin panel – manage users, uploads, view aggregated performance, and export reports.

Technology Stack

The platform is built with Python Flask, uses SQLite for data storage, and relies on Flask‑Login for authentication, Flask‑WTF for forms, Flask‑Mail for email notifications, and Bootstrap 5 for the frontend. Stock data is sourced via the yahooquery library.

Privacy & Security

All user data is stored securely within the application database. The platform is intended for internal use by the Prague Club of Investors and is not publicly accessible. Email addresses are used solely for authentication and notifications.
