How to export from Notion:
  1. Open Analysis Schedule DB
  2. Click ⋯ (three dots) in the top right → Export
  3. Select Markdown & CSV as the export format
  4. Select No files or images (we only need the CSV data)
  5. Click Export and download the ZIP file
  6. Extract the ZIP and upload the .csv file here
