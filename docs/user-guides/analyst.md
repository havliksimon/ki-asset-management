**How to read it:**
- **Positive spread** = You outperformed the benchmark
- **Negative spread** = Benchmark outperformed you

## ðŸ” Understanding Analyses

### Analysis Status

| Status | Meaning |
|--------|---------|
| **Scheduled** | Planned presentation |
| **On Watchlist** | Under review |
| **Approved** | Board voted to purchase |
| **Rejected** | Board voted against |
| **Other** | Non-stock event |

### How Returns Are Calculated

```
Return % = (Current Price - Price at Analysis) / Price at Analysis Ã— 100
```

**Example:**
- You analyzed Apple at $150
- Current price is $180
- Return = (180 - 150) / 150 Ã— 100 = **20%**

### Annualized Returns

For holdings over 1 year, returns are annualized:
```
Annualized Return = (Total Return + 1)^(1/Years) - 1
```

This allows fair comparison between short and long-term holdings.

## ðŸ’¡ Tips for Analysts

### Improving Your Win Rate

1. **Do thorough research** before presenting
2. **Consider multiple scenarios** (bull/bear/base cases)
3. **Set price targets** with clear rationale
4. **Monitor your picks** after presentation
5. **Learn from both wins and losses**

### Understanding Benchmarks

- **S&P 500** is the standard US benchmark
- **FTSE All-World** represents global markets
- Outperforming both is excellent
- Matching one is good
- Underperforming both suggests review needed

### Reading the Charts

- **Upward trend** = Improving performance
- **Downward trend** = Declining performance
- **Volatility** = Inconsistent results
- **Steady climb** = Consistent performance

## ðŸ†˜ Frequently Asked Questions

**Q: Why is my return different from what I calculated?**  
A: We use closing prices on analysis date and most recent available price. Slight timing differences are normal.

**Q: What counts as a "win"?**  
A: Any analysis with positive return percentage.

**Q: Why is my analysis not showing?**  
A: Only "Approved" status analyses count toward performance. Check with admin about board vote status.

**Q: Can I see other analysts' performance?**  
A: Admin can see all, but analysts typically see only their own. Check with your admin about visibility settings.

**Q: How often is data updated?**  
A: Stock prices update when admin runs recalculation (typically daily or weekly).

**Q: What's the difference between Average and Median return?**  
A: Average can be skewed by one very good or bad pick. Median shows the middle value and is often more representative.
