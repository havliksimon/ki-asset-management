## ðŸš€ Getting Started as a New User

### Registration Process

1. **Visit** the application URL
2. **Click** "Register" on the login page
3. **Enter** your `@klubinvestoru.com` email
4. **Check** your email for verification link
5. **Click** the link to set your password
6. **Log in** with your new credentials

### First Login

After logging in, you'll see:

**If you're an Analyst:**
- Personal dashboard with your performance metrics
- Access to blog (create/edit posts)
- Profile settings

**If you're an Admin:**
- Admin panel with full management access
- User management
- Data upload capabilities
- Performance tracking
- Board voting system
